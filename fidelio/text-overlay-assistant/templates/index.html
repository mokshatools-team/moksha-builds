First time? Initial setup ▼

Upload a video above and TOA will handle transcription and analysis. To place the overlays in DaVinci Resolve, set up the script and templates below.

1 — Install the DaVinci Resolve script

⬇ Download place_overlays.py

Copy to:

  • Mac: ~/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/Utility/
  • Windows: C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Scripts\Utility\

Restart Resolve. Script appears under Workspace → Scripts → place_overlays.

2 — Create Fusion Title templates

Create three templates with exactly these names (case-sensitive):

  • TOA_Chapter
  • TOA_List
  • TOA_Keyword

In each template, rename the Text+ node to OverlayText.

What do I do next? ▼

In DaVinci Resolve

  1. Open your project and make sure the correct timeline is active.
  2. Click Workspace → Scripts → place_overlays in the top menu bar.
  3. A file picker opens — navigate to Downloads and select the JSON file.
  4. The script places highlights on dedicated tracks:
    CH V4 — Chapter overlays LS V3 — List overlays KW V2 — Keyword overlays V1 — Your main edit, untouched
  5. Review the timeline, adjust timing or text, and render.
