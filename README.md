# Moksha Post Production

Shared builds for the Moksha tool ecosystem.

## Included tools

- `post-production-tool` - Social Scribe
- `brand-profile-tool` - Brand Profile Tool
- `cover-image-tool` - Cover Image Tool
- `content-inventory-tool` - Content Inventory Tool

## Notes

- Secrets and runtime files stay local via `.gitignore`
- The build brief for the cover tool lives in `COVER_IMAGE_TOOL_BRIEF.md`
